# kgarber.com

[kgarber.com](https://kgarber.com/)

Personal website generated using node.js, eleventy, markdown, and sakura.css.

Get started
```bash
npm install
```

Development server (served at http://localhost:8080/).
```bash
npx @11ty/eleventy --serve
```

Generate the website (output to `./docs`).
```
npx @11ty/eleventy
```

Publish to the web by pushing to master. GitHub pages serves the `/docs` folder.
